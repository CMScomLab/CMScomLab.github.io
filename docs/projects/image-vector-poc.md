# Image Vector PoC

画像のベクトル化技術を用いて、自然言語（日本語など）による画像検索の可能性を検証するプロジェクトです。

- **GitHub Repository**: [https://github.com/cmscom/image-vector-poc](https://github.com/cmscom/image-vector-poc)

## 概要

このプロジェクトでは、以下の技術的課題に取り組みます：

- 画像とテキストのマルチモーダルなベクトル化
- 自然言語クエリによる画像検索の精度検証
- 日本語環境における最適なモデルの選定

## 現在の状況

初期の検証として、Google SigLIP モデルを使用した画像検索システムのプロトタイプ開発を行っています。現状の比較検証では、日本語のクエリに対して SigLIP が比較的良好なマッチング結果を示しています。
