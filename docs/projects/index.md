# Projects

CMScomLabで進行中、または過去に実施したプロジェクトの一覧です。

## Active Projects

### [net-vis](net-vis.md)
JupyterLab向けのインタラクティブなネットワーク可視化ライブラリ。NetworkXのグラフ構造を簡単に可視化し、スタンドアロンHTMLとしてエクスポート可能です。

### [LSH Cascade PoC](lsh-cascade-poc.md)
ITQ-LSHとセグメントフィルタリングを組み合わせた、高速ベクトル検索エンジンの実装実験。HNSWと比較して40%の高速化とメモリ効率の改善を実証しました。

### [Image Vector PoC](image-vector-poc.md)
画像のベクトル化技術を用いて、自然言語（日本語など）による画像検索の可能性を検証するプロジェクト。現在はGoogle SigLIPを用いたプロトタイプを開発中。
